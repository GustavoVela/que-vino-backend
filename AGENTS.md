# README for Agents

Este documento proporciona contexto y reglas de comportamiento para todos los agentes de codificación (AI Coding Agents) que operen en este repositorio.

## Proyecto: Que Vino!?
**Propósito**: Plataforma de comunidad vinícola, e-commerce de vinos y asesoría experta con IA (Swarms).
**Arquitectura**: Micro-swarms independientes en Cloud Run.

## 🚩 Mandatorio: Constitución del Proyecto
Cualquier acción debe estar alineada con la **Constitución**: [.specify/memory/constitution.md](file:///.specify/memory/constitution.md). No violar bajo ninguna circunstancia.
